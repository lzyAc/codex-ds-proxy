使用说明

1
配置 DeepSeek API Key

在「配置」页面填入从 platform.deepseek.com 获取的 API Key

2
启动代理服务

代理默认在端口 8787 运行，状态指示灯变绿即表示工作中

3
设置终端环境变量 
export OPENAI_BASE_URL=http://127.0.0.1:8787/v1
export OPENAI_API_KEY=deepseek-proxy
4
运行 Codex

在同一个终端中直接使用 codex 命令，API 请求会自动走 DeepSeek
